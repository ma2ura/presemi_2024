固定効果モデル

固体効果，固体効果と年固定効果の両方を入れた二つのモデルを推定する。 固定効果モデルの推定には、fixestパッケージのfeols()関数を使うと便利

results <- list(
  "個体効果"      = feols(lwage ~ married + educ + exper + black + hisp | nr , data = wagepan),
  "個体・年度効果" = feols(lwage ~ married + educ + exper + black + hisp | nr + year, data = wagepan)
  )

  • 刑期の平均avgsen：マイナス
  • 人口当たり警察官の数polpc：マイナス
  • 人口密度density：プラス

で回帰する。
