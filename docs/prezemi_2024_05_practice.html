第5章 練習問題

第5章「パネルデータ分析」の練習問題を解いていきます。 結婚している人は結婚していない人に比べて賃金が高い（マリッジ・プレミアム）の要因分析を行います。

まずは事前の準備として，パッケージの読み込み，データの読み込み，文字化け対策を行います。

  • パッケージ: wooldridge, tidyverse, fixest
  • データ：wagepan.csv
# パッケージの読み込み
pacman::p_load(wooldridge, tidyverse, fixest)
# データの読み込み
data("wagepan")
# 文字化け対策
knitr::opts_chunk$set(dev = "ragg_png")

wagepanは44変数・4360観測値のデータセットです。 変数が多いので，以下で用いるデータのみ抽出し，dfに代入しておきます。

  • nr : 固体id
  • year : 年度
  • lwage : 賃金の対数
  • married : 既婚ダミー
  • educ : 教育年数
  • exper : 仕事経験年数
  • black : 黒人ダミー
  • hisp : ヒスパニックダミー
df <- wagepan |>
  select(lwage, married, educ, exper, black, hisp, nr, year)

(1) 回帰モデル

賃金の対数値lwageを，married，educ，exper，black，hispで回帰する。 プーリング回帰と固定効果モデルで推定する。教科書ではhissspとありますが，正しくはhispです。

プーリング回帰

年度効果と固体効果を無視したプーリング回帰分析を行います。

固定効果モデル

次に固体効果，固体効果と年固定効果の両方を入れた二つの固定効果モデルを推定します。 プーリング回帰と同様にlm()関数でも推定できますが，固定効果モデルの推定には便利なfixestパッケージのfeols()関数を使います。

results <- list(
  "個体効果"      = feols(lwage ~ married + educ + exper + black + hisp | nr , data = df),
  "個体・年度効果" = feols(lwage ~ married + educ + exper + black + hisp | nr + year, data = df)
  )
