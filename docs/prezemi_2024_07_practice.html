傾向スコアの作成

ロジスティックス回帰を使って，傾向スコアを作成します。

logit_model <- "training ~ age + age^2 + educ_cat + empstat_edt"
logit <- glm( # 一般化線形モデル
  logit_model,  # 回帰モデル
  data = df,  # データ
  family = binomial(link = "logit") # リンク関数
  )
summary(logit)
