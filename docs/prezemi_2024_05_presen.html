因果推論

相関関係と因果関係

  • 原因と結果の関係である因果関係(causal relation)があるのかどうか、どんな関係なのか、どの程度の強さがあるのかを推定することが因果推論です。
  • 相関関係(correlation)と因果関係は異なるもので、相関関係があるからといって、必ずしも因果関係があるわけではありません。
  • 同時決定性 と 逆の因果性

同時決定問題

  • 2つの変数が同時に変化することで、相関関係が生じること
  • 例えば、教育水準と所得の関係を考えてみましょう。
  • 教育水準が高い人ほど所得が高いという相関関係があるが、これは教育水準が高いから所得が高いのか、逆に所得が高いから教育水準が高いのか、が分かりません。
  • 相関関係は線形関係の強さを示すだけで、原因と結果の関係については何もいえません。

第3の要因

  • 2つの変数の間に因果関係があると考えられる場合、それらの変数に影響を与える第3の要因があるかどうかを考える必要があります。
  • 例えば、教育水準と所得の関係について考えてみましょう。
  • たとえば「ピアノを習っている」人ほど「学力が高い」という関係を見つけたとき、学力が高いことの原因がピアノを習っていたというより、隠れた要因として「親の所得」があるかもしれない可能性を検証する必要がある。
  • 疑似相関・見せかけの相関(spurious colleration)

第3の要因の図

  • 隠れた要因を，共変量(confounding variable)と呼ぶ。
  • 共変量\(C\)が\(X\)と\(Y\)の両方に影響を与える場合、\(X\)と\(Y\)の間に関係がなくても、\(X\)と\(Y\)の間に関係があるように見えることがある。

例題

キャリアセンターが集めたデータから、就活企画に参加した学生は、企画に参加していない学生と比べてその後の就職活動で成功している，ということが分かった。 そこから、就活企画に参加すると、就活に成功するという主張が可能でしょうか？

例題

キャリアセンターが集めたデータから、就活企画に参加した学生は、企画に参加していない学生と比べてその後の就職活動で成功している，ということが分かった。 そこから、就活企画に参加すると、就活に成功するという主張が可能でしょうか？

  1. キャリアセンター企画が就職活動の成功確率に直接影響している可能性
  2. 企画に参加した人はもともと就職活動に積極的な人で、企画に参加してなくても成功していた可能性
  3. 企画が就活の成否に全く影響を与えておらず、たまたま得られた結果

理想的な因果推論

  1. キャリアセンター企画に参加した自分と参加してない自分を比較する
  2. キャリアセンターに参加した自分は就職活動で成功
  3. キャリアセンターに参加してない自分は就職活動で失敗
  4. その差がキャリアセンター企画の効果であると結論づける

因果推論の方法

ランダム化比較実験 (RCT: Randomized Controlled Trial)

  • 生徒をランダムに2グループに分ける
  • 片方(処置群)のグループだけキャリアセンターの企画に参加してもらい、もう片方(対照群)は不参加
  • 一定期間後に2グループの就活結果を比較し、成功確率に差があるかどうかを検証

回帰分析の復習

線形回帰分析とダミー変数

  • 複数の説明変数と1つの被説明変数との間の(因果)関係を調べるための手法
  • 説明変数(ここでは\(X\))と被説明変数(ここでは\(Y\))との間に線形関係を仮定した線形回帰モデル
  • ダミー変数(ここでは\(D\))はカテゴリーを表す変数で0か1の値を取る変数

\[ Y_i = \alpha + \beta_1 X_i + \beta_2 D_i + \varepsilon _i, \]

  • \(\alpha\) 、\(\beta\) は回帰係数、\(\varepsilon_i\) は誤差項といい、特定の手法(OLSなど)で推定

線形回帰モデルの考え方

  • ある変数が他の変数にどのように影響を与えるかを推定

\(X\)と\(Y\)の関係を推定したら、正の関係があったとする。

共変量

  • 回帰分析において、説明変数以外の変数を共変量 (ここでは\(C\))と呼ぶ
  • 共変量\(C\)が\(X\)と\(Y\)の両方に影響を与える場合、\(X\)と\(Y\)の間に関係がなくても、\(X\)と\(Y\)の間に関係があるように見えることがある。

差の差分析

  • 差の差分析 (Difference-in-Differences: DiD) は、2つのグループにおいて、処置前後の変化を比較することで処置効果を推定する方法
  • パネルデータ (Panel Data, Logitudinal Data) は、同じ個体(unit)や地域における複数時点にわたって追跡調査したデータ

処置を受けた群を処置群(treatment)といい、\(Y_T\)で表す。 処置を受けていない群を対照群(control)といい、\(Y_C\)で表す。 グループに属する個体を\(i\)、時点を\(t\)で表す。 \(Y_{T,i,t}\)は処置群にいる\(i\)の\(t\)時点のデータを表す。

差の差の分析

処置群と対照群の結果の差を比較するため、

\[ \underbrace{\underbrace{( Y_{T,i,t} - Y_{T,i,t-1} )}_{処置群の差} - \underbrace{( Y_{C,i,t} - Y_{C,i,t-1} )}_{対照群の差}}_{差の差} \]

を計算する。 これが差の差の分析(Difference in Difference: DiD)の基礎概念である。 ただ、これはほぼ不可能なので、代わりに次のように推定する。

反実仮想

理想的な因果推論の方法として、反実仮想(counterfactual)を考える。 先のキャリアセンター企画の例で言うと、

  • 自分\(i\)がキャリアセンター企画に参加したら、参加しなかった自分はいなくなる。
  • しかし本当に知りたい処置効果は、参加した自分と参加しなかった自分の差である。
  • 処置群の\(Y_{T,i,t}\)に対して、対照群の\(Y_{C,i,t}\)は反実仮想である。

反実仮想の推定

  • 反実仮想は観察不可能なので、反実仮想を推定する方法が必要
  • その方法が平均処置効果(ATE: Average Treatment Effect)の推定方法
  • 個々の処置効果は観察不可能だけれど、処置群と対照群の平均の差を比較することで、処置効果を推定できる。
  • ランダムに処置を振り分けられている場合、平均をとることで個々の要因を相殺することができる、と考える。

回帰モデル

差の差分析は、回帰モデルを使って推定することができる。

\[ \begin{aligned} Y_{i,t} = \alpha &+ \beta_1 Treat_i + \beta_2 After_t \\ &+ \beta_3 (Treat_i \times After_t) + \beta_4 X_{it} + \varepsilon_{i,t} \end{aligned} \]

  • \(Treat_i\) : 個体\(i\)が処置群なら1、そうでないなら0
  • \(After_t\) : 時点\(t\)が処置後なら1、そうでないなら0
  • \(Treat_i \times After_t\) : 処置群かつ処置後なら1、そうでないなら0

回帰係数の解釈

\[ \begin{aligned} \frac{\partial Y_{i,t}}{\partial Treat_i} &= \beta_1 + \beta_3 After_t \\ \frac{\partial Y_{i,t}}{\partial After_t} &= \beta_2 + \beta_3 Treat_t \\ \frac{\partial^2 Y_{i,t}}{\partial Treat_i \partial After_t } &= \beta_3 \end{aligned} \]

ダミー変数の作成

DiDで用いる処置群ダミーと処置年度ダミーを作成する。 前回と同じよう、dplyr::mutate()とdplyr::if_else()を使って作成する。

df <- df |>

回帰分析でDiD

\[ Y_{it} = \alpha + \beta_1 Treat_i + \beta_2 After_t + \beta_3 Treat_i \times After_t + \beta_4 X_{it} + \varepsilon_{it} \]

\(\beta_3\)は

パネルデータのDiD

パネルデータとは？

  • 複数の個体\(i\)を、複数の時点\(t\)で観測したデータをパネル・データという。
  • パネルデータの利点は、
    1. 個体\(i\)がもつ特徴をコントロールできる。
    2. 時間による変化を評価できる
    3. サンプル・サイズが大きくできる

固定効果モデル

固定効果とは？

  • パネルデータは個体\(i\)ごとに時系列で追跡できる特性を持っている。
  • 都道府県を\(i\)、時点を\(t\)とし、各都道府県の母就業率\(Y_{it}\)を考える。
  • 各都道府県\(i\)の保育所定員率\(X_{it}\)と時間で変わらない都道府県の特性\(\mu_i\)で決まるとする。

\[ Y_{it} = \alpha + \beta X_{it} + \mu _i + \varepsilon _{it} \]

この\(\mu_i\)をどうやって設定するかが問題。

固定効果の測定方法

  • ダミー変数法 : 各都道府県にダミー変数を設定し、都道府県ごとの特性を推定する。
  • 47都道府県なら、46個のダミー変数を設定する。つまり

\[ \begin{aligned} Y_{it} = \alpha &+ \beta_1 X_{it} + \beta_2 北海道ダミー + \beta_3 青森ダミー \\ & + \beta_4 秋田ダミー + \cdots + \beta_{47} 鹿児島ダミー + \varepsilon _{it} \end{aligned} \]

この各都道府県ダミーを固定効果(fixed effect)といい、観察できない都道府県の特性を表していると考える。

時点固定効果

  • 時点ダミーを設定することで時点ごとに異なる特性を推定できる。
  • 経営研究では、年度ダミーを設定することが多い。
  • たとえば2010年から2024年の25年間のデータを使う場合、24個の年度ダミーを設定する。

\[ \begin{aligned} Y_{it} = \alpha &+ \beta_1 X_{it} + \beta_2 2010年ダミー + \beta_3 2011年ダミー \\ & + \beta_4 2012ダミー + \cdots + \beta_{47} 2023年ダミー + \varepsilon _{it} \end{aligned} \]

二次元固定効果モデル

  • 個体\(i\)がもつ固定効果と、時点\(t\)がもつ固定効果を同時に推定するモデルを二次元固定効果モデルという。
  • 近年の実証研究では標準的な推定方法となっている。

\[ \begin{aligned} Y_{it} = \alpha & + \beta_1 X_{it} + \sum _{k = 2}^{24} \beta_k 年度ダミー_k + \sum _{l = 25}^{47} \beta_l 個体ダミー_l + \varepsilon _{it} \end{aligned} \]

DiDとしての固定効果モデル

固定効果モデル

パネルデータは2時点\(t=1\)と\(t=2\)をもつ。

\[ \begin{aligned} Y_{i,1} &= \alpha + \beta X_{i,1} + \gamma _1 D_1 + \cdots + \gamma _k D_k + \varepsilon _{i,1} \\ Y_{i,2} &= \alpha + \beta X_{i,2} + \gamma _1 D_1 + \cdots + \gamma _k D_k + \varepsilon _{i,2} \end{aligned} \]

\(Y_{i,1} - Y_{i,2} \equiv \Delta Y_i\)をとると、

\[ \begin{aligned} Y_{i,1} - Y_{i,1} &= (\alpha - \alpha) + (X_{i,1} - X_{i,2}) \beta \\ &+ (\gamma_1 D_1 - \gamma_1 D_1) + \cdots + (\gamma_k D_k - \gamma_k D_k) + (\varepsilon_{i,1} - \varepsilon_{i,2})\\ \Delta Y_i &= \beta \Delta X_i + \Delta \varepsilon_i \end{aligned} \]

階差回帰モデル

  • 差分をとることで時点間で変化しない定数項と個体ダミー(\(\gamma_i\))は消える。
  • \(\beta\)は\(X\)が1変動した\(\Delta X\)に対する\(Y\)をどれだけ変化させるかを示す。
  • \(X\)が全く変化しないのを比較群、1変化したのを処置群とすると、係数\(\beta\)は処置群と比較群の\(Y\)の変化幅の差を示す。つまりDiD
\(X\)の変化 \(Y\)の変化
処置群 \(\Delta X = 1\) \(\beta\)
比較群 \(\Delta X = 0\) \(0\)

パネルデータのDiDの特徴

  1. 処置群ダミーと処置後ダミーの交差項のみをモデルに含める
  2. 時間を通じて変化しない変数は説明変数に含められない
  3. FEモデルによるDiDでは説明変数として導入できない第3の要因を考慮できる

パネルデータ分析に時間を通じて一定の変数を導入するには、相関変量効果(correlated random effects)モデルを使う。
